FileFusion

File Concatenation Tool Optimized for LLM Usage

Concatenate files into formats optimized for Large Language Models while preserving metadata

Download View on GitHub

Installation

# Using Go install (recommended)
go install github.com/drgsn/filefusion/cmd/filefusion@latest
# Manual installation
git clone https://github.com/drgsn/filefusion.git
cd filefusion
go build ./cmd/filefusion
# From releases - download and add to your PATH
curl -L https://github.com/drgsn/filefusion/releases/latest/download/filefusion_linux_amd64.tar.gz | tar xz
sudo mv filefusion /usr/local/bin/

Pre-built binaries are available for Linux, macOS, and Windows on the releases page.

Features

Multiple Output Formats

Export your concatenated files in XML, JSON, or YAML formats for maximum compatibility

Pattern Matching

Use glob patterns to include or exclude files based on your needs

CLI Interface

Simple command-line interface for easy integration into your workflow

Size Control

Built-in file size limits and output size control to prevent processing oversized files

Examples

# Basic usage - process all Go files in current directory
filefusion --pattern "*.go" --output result.xml .
# Multiple patterns and exclusions
filefusion --pattern "*.go,*.json" --exclude "vendor/**,*.test.go" --output api.json ./api
# Size limits and YAML output
filefusion --max-file-size 5MB --max-output-size 20MB --output docs.yaml ./docs
# Process multiple directories
filefusion --pattern "*.go" ./cmd ./internal ./pkg
